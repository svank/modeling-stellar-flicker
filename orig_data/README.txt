The data files in this directory derive from the following sources. Data from any third-party sources are in no way covered by the LICENSE file of this repository.

Note that only data files required to run any function in `base.py` are included in this repository. Other data files (those loaded directly by a Jupyter notebook, or those used by `build_catalog.py` to build our merged catalog) are included in our Zenodo repository at https://doi.org/10.5281/zenodo.4444282

-------------------------

aesopus/

The contents of this directory come from the web interface at http://stev.oapd.inaf.it/aesopus which is maintained by Paola Marigo and hosted at the Osservatorio Astronomico di Padova.
The file names indicate the value of Z_ref used, with all other input parameters the same from one file to another. All input parameters are listed in the headers of the files.


bandpass_adjustment.txt

This file is bandpass_correction/step2_output.txt (relative to the root of this repository) with some empty lines removed. See the bandpass_correction directory for details.
